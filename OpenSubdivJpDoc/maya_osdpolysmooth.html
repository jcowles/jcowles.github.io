DESCRIPTION

The Maya OsdPolySmooth node takes a base mesh and generates an OpenSubdiv subdivided mesh.

Note: This plugin is not representative of actual workflows in the application. The intended use is for quick comparisons with Renderman's interpolation settings.

Usage

Once the plugin is loaded in Maya:
  • Select a poly mesh
  • Polygons -> Edit Mesh -> OSD Subdivide Meshes
